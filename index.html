HTML

  • The head element contains information about the webpage.
  • The title element defines the title for the webpage.
  • The style element defines the css styles associated with a webpage.
  • The base element defines the base URL for a webpage.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport
  • The script element embes client-side scripts in the html document.
  • The carset element sets our character encoding.
  • The http-equiv element sets an HTTP header for content.
  • The viewport element sets the default view for a webpage.
  • The body element represents the visible content shown to the user.
  • The div element helps to create a division in the page.
  • The span element creates an inline division.
  • The h1 (h2),(h3) element represents the leavel of heading and given text block.
  • The p element represents paragraph or block of text.
  • The img element contains information about images that are displayed on the webpage.
  • The br element creates a line of empty space or a line break between two blocks of content.
  • The a element creates links to the same webpage, or other webpages.
  • The element section element are semantic elements that define a section in a document that contains thematic content.

CSS

  • A selector defines the element or attributes to which the rules, or declarations will apply.
  • Declarations contain two important components: the css property we want to apply and the value of the property.
  • The link element creates the conection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) od the external resource.
  • The Margin property indicates how much space we want around the outside of a element and is not visible on the page.
  • The padding property add space around the content inside an element.
  • The border-style property specifies what kind of border to display.
  • The box model allows us to add a border around elements, and to define space between elements.
  • The outline-style property specifies the style of the outline.
  • p is a selector in CSS (it points to the HTML element you want to style).
  • Class attributes allow us to share a css rule to any elemnt we choose by assigning the rule to a class attribute with a class selector.

Git

  • Git is the free and open source distributed version control system tghat's responsible for everything GitHub related that happens locally on your computer.
  • Git add Moves changes from the working directory to the staging area.
  • Git config --global user.name set a name that is identifiable for credit when review version history.
  • Git add [file] add a file as it looks now to your next commit.
  • Git clone Creates a copy of an existing Git repository.
  • Git config A convenient way to set configuration options for your Git installation.
  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it
  • Git commit -m "name" staged files to be committed
  • Git pull origin main we are receiving a branch called main from the GitHub repository.
  • Git push origin push the changes we have locally to our remote GitHub branch.
  • Git branch list your branches.
  • Git merge [branch] merge the specified branch's history into the current one.
  • Git log show all commits in the current brach's history.
  • Git remote add [alias] [url] add a gir URL as an alias.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • To declare a variable we use the var keyword
  • We give the variable a name that must be unique.
  • We use an assignment operator = to assign the specific data that we want to stored.
  • The console.log method can be use to output a message to the web console by an argument.
  • A truthy value is a value that is considered true when encountered in a Boolean context.
  • Control flow is the order in which a computer executes code in a script.
  • The code runs from the first statement at the top of the page, down to the last statement at the botom of the page.
  • We can use conditionals and loops to control the order in which code is executed, or if it is even executed at all.
  • Conditionals are code structures used to test if an expression returns true or not.
  • The if statement is a way of introducing decision-making into our code.
  • The else if statement is use when we want to provide more conditions than just if.
  • An array is a single variable that is used to hold a group of data that are related in some way.
  • To create an array we use var and then give the variable a name and we assign the variable to a group os values.
  • The for loop uses the predictable pattern of indices to perform a taskon all the items in any array.
  • A for loop allows a single code block to be executed over and over.
  • A for loop contains the following three important statements: fisrt statement determines the starting point for our loop, we declare a variable x and give it a values of 0. Second statement is the condition. As long as the number in the variable x is less thanthe length of the array, the loop will keep runnning. Third statement is what allows the array iterate over each time (The code x++ is a shortcut way of writing).
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • Math function help us select a random number.
